# 🟦 Block Dodger

> 落ちてくるブロックを避け続ける一画面アボイダー

左右に動いて落下するブロックをかわし、できるだけ長く生き延びるシンプルな回避ゲームです。時間が経つほど難易度が上がっていきます。

## 🎮 操作方法
| 操作 | 動作 |
|---|---|
| 左 / 右 | 左右に移動してブロックを回避 |

---

## ✨ 特徴
- **一画面で完結** — ルール説明不要のシンプルな回避ゲーム
- **左右移動だけ** — 直感的な操作性
- **難易度上昇** — プレイ時間に応じて徐々に難しくなる
- **サバイバル** — どれだけ長く生き延びられるかを競う
